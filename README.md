# The Experience with Django 
## The Experience is a website for users to review songs, a lot like Letterboxd, and to read articles about pressing musical issues. It's a new experience to listen to music.

# How to run raw files 
First, you must create a virtual environment with python -m venv [environment name]. It can be anything that you wish 
To activate it depends on your operating system. 
On Linux or Mac, it is: source [environment name]/bin/activate

On Windows, it is: [environment name]\Scripts\activate.bat, however, on my Windows machine, I use: [environment name]\Scripts\activate.ps1

Now you must install Django before cloning the project
Install Django with: python -m pip install Django 

After this you can clone the project and run the program as normal 
The command to run the Django program is: python manage.py runserver
